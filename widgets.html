Select¶

class insanities.forms.widgets.Select([size][, null_label][, **kwargs])¶

Takes options from EnumChoice converter, looks up if converter allows null and passed this value as template required variable.

null_label¶
Label assigned to None value if field is not required
size¶
HTML select element’s select attribute value.
