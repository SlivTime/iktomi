Module Description¶

BaseField¶

class insanities.forms.fields.BaseField([label][, template][, media][, **kwargs])¶

Simple container class which ancestors represents various parts of Form.

Encapsulates converter, various fields attributes, media objects, methods for data access control, widget to render the g

classname¶
Passed to widget and rendered as HTML element of field’s classname. Can be set by field inheritance or throught constructor.
env¶
Field’s form environment
error¶

Field’s error in last validation

The property is available only if field is bound.

form¶
Field’s form
id¶
Field’s input ID in order of form’s ID and field’s input name.
input_name¶

Name of field’s input element generated in account to possible nestiong of fields. The input name is to be used in templates as value of Input (Select, etc) element’s Name attribute and Label element’s For attribute.

It is also used as key in form’s data multidict.

The property is available only if field is bound.

label¶
Label of field. Can be set by field inheritance or throught constructor.
media¶
FormMedia incstance containing media objects related to the field. Can be set by field inheritance or throught constructor.
multiple¶
Proxy property. Checks if field’s converter is multiple
parent¶

Proxy getter of current field’s parent. Parent can be Form instance or other BaseField instance if there are field nesting.

Raises FormError if it is attempt to access the parent of an unbound field.

perm_getter¶
FieldPerm instance determining field’s access permissions. Can be set by field inheritance or throught constructor.
permissions¶
Returns field’s access permissions
value¶
Current field’s converted value from form’s python_data.

Field¶

class insanities.forms.fields.Field([name][, conv][, widget][, **kwargs])¶

Atomic field

accept()¶
Converts field’s raw value to python, but raises SkipReadonly exception if there are no write permission to the field.
conv¶
Conv subclass or instance used to convert field data and validate it
fill(data, value)¶
Sets the value in given multidict using field’s input_name as key.
get_default()¶
Returns fields default value
get_media()¶
Returns a list of field’s media objects included ones from field’s widget.
grab()¶
Returns field’s raw value from form’s data multidict.
render()¶
Renders the field.
widget¶
Widget subclass or instance used to render the field. Can be set by Field inheritance or throught constructor.
